为什么不用固定价格套餐？

每次克隆任务的成本差异极大——静态落地页和复杂 SaaS 后台消耗的 AI Token、代理请求量完全不同。固定套餐要么让平台亏损，要么让用户多付。

我们选择动态定价：系统精确计量每个成本项，任务完成后乘以利润率系数收费，账单透明到每一个 API 调用。

最终价格 = 实际成本 × 利润率系数 // 实际成本 = Decodo 代理 + Claude Token + Playwright + 容器测试 // 利润率系数默认 5x，专业订阅用户 3.5x

常见问题
什么是"实际成本"，如何计量？
每次克隆任务中，系统精确记录所有 API 调用：Decodo 代理请求数、Firecrawl 页面数、Claude API 的输入输出 Token 数、Playwright 和 Docker 运行时长、Cloudflare R2 存储用量。任务完成后汇总为实际成本，乘以利润率系数得出最终账单，明细会发送到邮箱。
克隆失败怎么办，钱会退吗？
系统自动重试最多 3 次。若仍失败，本次任务费用全额退款到支付方式。由于我们采用"完成后扣款"模式，通常你实际上不会为失败的任务付费——只有成功完成的任务才会扣款。
支持哪些网站登录方式？
通过平台内嵌浏览器，支持任意登录方式：账号密码、短信验证码、Google/Apple OAuth、企业 SSO、图片 CAPTCHA。用户在内嵌浏览器中亲自完成登录，平台提取 Session Cookie 后继续抓取，不额外存储你的账号信息。
月度订阅和按次计费怎么配合使用？
月度订阅仅降低每次克隆的利润率系数（默认 5x 降至 3x-4x），不包含克隆次数。每次克隆仍按实际成本 × 订阅后的系数计费。高频用户订阅后，复杂网站每次可节省 $5-18。
